# Setup

### Install dependencies

```
git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim

:PluginInstall

:GoInstallBinaries

# install ctags (gotags, vim-go)

go get -u github.com/jstemmer/gotags
```
